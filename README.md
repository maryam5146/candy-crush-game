# Candy Crush–Style Puzzle Game

## Description
This project is a Candy Crush–style puzzle game developed as part of a programming project. It implements core gameplay mechanics such as grid-based movement, match detection, and score tracking.

## Technologies Used
- Programming Language: C++ 
- Concepts: Programming Fundamentals, Problem Solving, Logical Thinking

## Features
- Grid-based game board
- Match detection logic
- Score calculation
- Interactive gameplay

## Purpose
This project was created to strengthen understanding of programming fundamentals and problem-solving through a practical game-based implementation.
